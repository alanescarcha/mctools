About mctools.pro

About

mctools.pro is a site that collects useful tools for server administration and developers.

Some of the tools were created by other people, clicking on the tool will take you to the developer's page, some others were modified from open source code, other tools were created by me from scratch using external APIs and Libraries.

GitHub Repo: https://github.com/alanescarcha/mctools

Libraries and APIs used
- Bootstrap v5.1 - Api-Mojang v1.1 (created for this project) - SweetAlert2 - Axios - API-MCSRVSTATUS - PlayerDB - Crafatar
